## Убираем надпись об окончании лицензии.
1. Ищем файл bitrix\modules\main\includ.php
2. Проводим его Деобфускацию с помощью php скрипта phpCB. Команда: "phpCB.exe include.php > normalize.php"
3. Примерно в 140 строке ищем код "$GLOBALS[___311817900(119) ] = OLDSITEEXPIREDATE;".
4. Меняем "OLDSITEEXPIREDATE" на "time()+86400*1".
5. Комменим цикл for в, примерно, 137 строке (for ($_2025645906 = (162 * 2 - 324), $_648911838 = ($GLOBALS['____1871159341'][65]() < $GLOBA..........).
6. Лицензия становится бесконечная, но без обновлений. Битрикс пишет что остался 1 день, но на самом деле он никогда не кончится.

Еще можно получить демо-ключ тут https://www.1c-bitrix.ru/bsm_register.php
